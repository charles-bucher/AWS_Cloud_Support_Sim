## About
This repository simulates a **Cloud Support engineering workflow** on AWS.  
It documents **real-world troubleshooting scenarios**, following the **support ticket lifecycle**: from customer report, through root cause analysis, to resolution.  
All steps are supported with screenshots for **authenticity and proof of hands-on skills**.

---

## Skills Demonstrated
- AWS EC2, VPC, Security Groups, IAM, S3
- Terraform infrastructure provisioning
- Python automation & monitoring scripts
- Git version control and workflow management
- Incident response, troubleshooting, and operational excellence
- Documentation and scenario walkthroughs
